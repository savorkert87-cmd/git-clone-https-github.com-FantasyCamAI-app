## ✨ Fonctionnalités

### Web
- 🎥 Interface web de gestion des vidéos
- 🧠 Analyse IA en temps réel

## 🛠️ Stack technologique

### Backend
- **Python 3.9+**
- **FastAPI** - API REST
- **SQLAlchemy** - ORM
- **PostgreSQL** - Base de données
- **Redis** - Cache et queue
- **OpenCV/MediaPipe** - Vision par ordinateur
- **TensorFlow/PyTorch** - Modèles IA

### Frontend Web
- **React** ou **Vue.js**
- **TypeScript**
- **Tailwind CSS**
- **Axios** - Client HTTP

### Mobile
- **React Native** ou **Flutter**
- **SQLite** - Base de données locale
- **Redux** - State management

## 🚀 Installation

### Prérequis
- Python 3.9+
- Node.js 16+
- PostgreSQL
- Redis

### Backend

```bash
git clone https://github.com/savorkert87-cmd/git-clone-https-github.com-FantasyCamAI-app.git
cd git-clone-https-github.com-FantasyCamAI-app

python -m venv venv
source venv/bin/activate

pip install -r requirements.txt
cp .env.example .env
```

### Frontend Web

```bash
cd web
npm install
npm start
```

### Mobile

```bash
cd mobile
npm install
npm start
```
